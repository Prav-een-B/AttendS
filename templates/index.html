How it works — local pipeline
01
Preprocess
OpenCV deskews, denoises & binarizes the photo using adaptive thresholding.
02
Table Detection
Morphological ops find row & column lines. Falls back to horizontal-band segmentation for hand-ruled sheets.
03
Tesseract OCR
Each cell is individually OCR'd with optimised PSM modes for names, numbers, and status marks.
04
Parse & Export
Heuristics map columns to roll/name/status. Output is a formatted, colour-coded Excel file.
